## 🎯 Why AI Resume Analyzer Pro?

Most resume tools just check grammar. This one thinks like a recruiter.

Built with cutting-edge AI, this tool goes beyond surface-level feedback — it evaluates ATS compatibility, surfaces skill gaps, generates role-specific cover letters, and even coaches you on interview questions tailored to your background.

Whether you're a fresh graduate or a senior engineer switching roles, **AI Resume Analyzer Pro** gives you an unfair advantage.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) v18 or higher
- [Bun](https://bun.sh/) *(recommended)* or npm

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/Monish892/Resume-Analyzer.git

# 2. Navigate to the project
cd Resume-Analyzer/resume-ai-pro

# 3. Install dependencies
bun install        # or: npm install

# 4. Start the development server
bun run dev        # or: npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser — that's it!
